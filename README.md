# tad_Maior

#### Exercício - TAD's e objetos C++

A estrutura de dados contém os seguintes métodos com as respectivas complexidades:
- Inserir: O(1)
- Buscar maior: O(1)
- Ordenar: Selection sort, O(n²)
- Tamanho: O(1)
- Imprimir: O(n)

O algoritmo também possui um método construtor e um destrutor.
